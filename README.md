## Overview

Faustop programming language is a language built in java, for academic (kk) porpuses. 

## Documentation

### 1 Data types
| Faustop type        | Java equivalent       |
|:-------------------:|:---------------------:|
| `olokinho`          | `int`                 |
| `oloko`             | `double`              |
| `bicho`             | `String`              |
| `paiseuropa`        | `boolean`             |
